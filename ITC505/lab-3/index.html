ITC 505 – Lab 3

To deliver a custom font to a browser, I used the Google Fonts service. The font files are hosted on Google’s servers. By including the provided <link> tag inside my HTML <head>, the browser automatically downloads the font. In my CSS, I applied the font-family to style this page. This ensures that even if the font is not installed on the user’s computer, the page still displays in the chosen style.
